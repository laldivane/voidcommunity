# 🌌 Lal Divane - Infinite Void Platform

Lal Divane is a premium, high-fidelity digital music experience built for the infinite digital abyss.

## ✨ Features
- **Archives (Albums)**: Explore curated manifestations like "Rituals of the Digital Void".
- **Real-Time Presence**: See what the void is currently listening to.
- **Community Hub**: Manifest thoughts and engage with the digital collective.
- **Premium Audio**: High-fidelity raw links integrated directly from the digital essence.

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS (v4), Framer Motion.
- **Backend**: Supabase (Auth, Database, Realtime).
- **Styling**: Glassmorphism, HSL tailord crimson themes, and custom shaders.

## 🚀 Deployment
Deployed on **Vercel**. 
Ensure the following environment variables are set:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 📦 Setup
```bash
npm install
npm run dev
```
